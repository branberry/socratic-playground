# rust-webgpu

WebGPU learning track — **scaffold only** for now.

Same monorepo principles as [rust-rag-learn](../rust-rag-learn/): Socratic tutoring, small steps, you write the code.

## Quick start

```bash
cargo test -p rust-webgpu
cargo run -p rust-webgpu
```
